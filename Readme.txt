#Readme
- To change project name/context edit property 'server.context-path' in application.properties

#Tomcat Server
- To change Tomcat running port edit property 'server.port' in application.properties

#Tools Integrated
- Spring boot
- Spring boot embedded Tomcat
- Thymleaf
- Spring Data Jpa (Hibernate as the implementation and ORACLE as the database)
- Flyway
- Spring security OAuth2

#Updates new
- Added spring integration support.

- Added a simple HttpOutBound channel adapter for hitting a web service.
- Added a simple design with couple of pages.
